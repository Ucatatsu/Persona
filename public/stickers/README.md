# Стикеры Квант

Эта папка содержит анимированные стикеры в формате .tgs для мессенджера Квант.

## Формат .tgs
- .tgs файлы - это сжатые Lottie анимации
- Используются в Telegram как анимированные стикеры
- Поддерживают прозрачность и высокое качество анимации

## Структура
- Поместите .tgs файлы в эту папку
- Файлы будут автоматически загружены в стикер-пикер
- Рекомендуемое количество: до 1000 стикеров

## Использование
1. Загрузите .tgs файлы в эту папку
2. Перезапустите сервер или обновите кэш
3. Стикеры появятся в стикер-пикере рядом с эмодзи
